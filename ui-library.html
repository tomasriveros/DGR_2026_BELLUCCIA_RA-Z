08 — Espaciado y estructura

  • --max-w: 1240px — ancho máximo de contenido centrado
  • --space-section: clamp(3rem, 6vw, 5rem) — padding vertical estándar de sección
  • --border-w: 2px — grosor de borde estándar en tarjetas y botones
  • --shadow-sm / md / lg — sombra tipo "cómic" desplazada, escalable en 3 tamaños
